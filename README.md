# First PCB Design Project using EasyEda Designer and Learned By Altium Education

## Overview

From knowing only the basics of what a PCB is to designing one from scratch, this project marks an important milestone in my electronics learning journey.

This project was created using Easyeda Designer as part of the Altium Education PCB Basic Design Course. The objective was to understand the complete PCB design workflow, including schematic creation, PCB layout design, routing, validation, and manufacturing file generation.

---

## Project Title

**Power Regulator and LED Board**

---

## Objectives

- Learn PCB design fundamentals
- Create a schematic from circuit requirements
- Design a PCB layout
- Perform component placement and routing
- Validate the design using DRC
- Generate Gerber files for manufacturing
- Visualize the design in 3D

---

## Tools Used

- EasyEda Designer
- Altium Education Learning Platform

---

## Design Workflow

### 1. Schematic Design
Created the circuit schematic for the Power Regulator and LED Board.

### 2. PCB Layout
Transferred the schematic to PCB and arranged components for an efficient layout.

### 3. Routing
Connected all electrical nets through PCB traces.

### 4. Design Rule Check (DRC)
Verified the design and resolved routing and clearance issues.

### 5. 3D Visualization
Generated a 3D representation of the PCB to inspect the final design.

### 6. Gerber Generation
Generated manufacturing files required for PCB fabrication.
